Traffic Simulator Configuration Tool, by Stefan79 and z

The Traffic Simulator Configuration Tool is a program that will allow you to tune your traffic simulator and the NAM data views to your liking.  It can be invoked either during the NAM installation, or separately at any time.
 Screenshots

 Information

Full information on using the Traffic Simulator Configuration Tool can be found in its User Guide, which is accessible through the Help menu (visible above).  Of particular interest to users of older NAM traffic simulators is the section near the end of the User Guide entitled Using the Traffic Simulator Configuration Tool to Emulate Older NAM Simulators. This section will tell you the best settings to use if you want to emulate one of these older traffic simulators.
 Compatibility

This download is only compatible with SimCity 4 "Rush Hour" or SimCity 4 Deluxe, and with Simulator Z or the NAM Unified Traffic Simulator (the NAM Simulator).
 Installation

Auto Installation: Installation of this program is started automatically by the NAM Installer as needed. This program's installer will guide you through the installation process. You can change the default installation path if you wish.

MAC Users: File Juicer should enable you to use this installer package.

Uninstalling: To uninstall the files, Windows users should run the program Uninstall.exe from the program's installation directory, or use Add/Remove Programs in the Control Panel.  Mac users should simply remove the installation directory and its contents.
