You can make modifications to the lot files by moving them into your main My Documents\SimCity 4\Plugins directory and simply using the Lot Editor as normal. Installation
(A basic how-to and a note on executable/EXE installers)
 

In order to install this download, all you need to do is unzip all of the files contained to any subdirectory of the folder My Documents\SimCity 4\Plugins. It does not matter which subdirectory you choose, although I recommend putting it under a "NDEX" folder so you can easily access it if you want to delete, edit, or alter it in the future.

Some members of the Simtropolis community who release many items on the Exchange have decided to switch to using executables (Windows app or program, .exe files) to distribute their creations. Whereas this is their choice, I feel this is rediculous, unsafe for the user (even though the members doing this are trusted, for the most part, the threat of an installer placing a virus on your computer is very real), and also unfair to players of the Macintosh version of the game, who cannot make any use of a Windows application file. Therefore, NDEX will never resort to this method of distribution. We hope to keep our creations open to as many people as possible, and if you can't unzip an archive file, you wouldn't have gotten this far.

 


Note about Custom Model Nightlighting Effects
 

In order to see night lighting, you need to have the EP1 Update 1 for Rush Hour/Deluxe installed, and then afterwards install the patch specifically for the BAT nightlighting, available on the "Building Architect Tool" page on the official Maxis website.  It is a separate download because it is optional and would add 19 MB to the other update.

If you do not have this installed, no custom model you download will display
any lighting effects at night.
