PRODUCT DESCRIPTION


 

The CDK3-SP Container Dock is similar to the Container Seaport in appearance. But it is not a functional seaport. Its designed purpose is to visually extend your container processing operations in and around your seaport complex.

Slightly smaller at only 9 tiles wide, the cental rail connection has been replaced with a single, centered street entrance. The number of cranes have been reduced from three to two so the lot can be easily discerned from a functional seaport. As multiple instances of this lot are likely to be used, the lot uses more random truck, rail car and container props to add more visual variety.

The lot is technically a ploppable IND-D lot with jobs, costs and properties similar to the CDK3 Vandy Trucking lot. Because of the jobs, the lot requires road access.


Transit Enabled Driveway
The lot's driveway is transit enabled as a street to avoid forcing an intersection. The transit enabling is visual only... to allow a visual connection to the adjacent road.  The lot will not allow pedestrian or vehicle traffic to pass through it.


Transit Enabled Rail
The rail running through the lot is fully transit enabled to allow both passenger and freight rail to pass through the lot. This is an optional feature to give the player greater flexibility in using the lot. However, the lot itself does not require functional rail in any way and in most applications, the functional aspect of the rail can simply be ignored.


Lot Spacing
These lots can not be plopped adjacent to each other. The ships are slightly longer than the lot and would "smush" together.  The CDK3-SP 1x Lots are ideal for adding the required spacing between any of the lots that have ships.

INSTALLATION

If installing from the ZIP archive, download the file to your C:\My Documents\SimCity 4\Plugins folder. Then, unzip the file making sure that the "Use Folder Names" option is selected. The product will be installed in an appropriate folder under the PEGPROD sub-folder which will be created if it does not already exist on your system. Once installed, the product will be ready to use in your game. Lots will appear in the menu listed in the above Product Details section.
** After installation, you can either delete the downloaded ZIP file or archive it outside of your Plugins folder satructure.
DEVELOPER NOTES

The lot is a CDK3 lot and therefore must be plopped over water. It has been designed to have the inner most 3 rows of tiles plopped fully on dry land. 

The lot has a driveway entrance that is transit enabled to make a visual connection only. The lot itself is not transit enabled. It will connect automatically to an existing road network.  If the road network is added after the lot is plopped, you can connect it manually using streets. This connection is only visual. It is not required for lot functionality. However, the lot must have road access in order to function properly as it has jobs..


The rail running through the middle of the lot is fully transit enabled for both passenger and freight rail. This was added to give the player the option of connecting the lot to the city's rail network and/or allowing functional rail to pass through the lot. However, the rail is not used by the lot itself in any functional way. For most applications, it can be ignored completely.


You can not plop these lots adjacent to each other. The ships are slightly longer than the lots and will be too close to each other. It is recommended to place one or two of the CDK3-SP 1x Lots between these lots... or other lots with ships or transit enabled rail. 

The lot must be plopped with the 3 inner rows of tile on dry land. It is often helpful to use the CDK Leveler tool before plopping CDK lots.
