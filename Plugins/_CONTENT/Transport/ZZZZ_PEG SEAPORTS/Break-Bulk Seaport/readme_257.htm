PRODUCT DESCRIPTION


 

The CDK3-SP Break-Bulk Seaport is a functional seaport that is identical in all regards to the CDK3-SP Container  Seaport. The size of the lot, plop & maintenance costs... and road & rail connections are all the same.

The only real difference in the lot is visual. This version of the seaport is designed to blend well with the CDK3-SP Row Warehouses  and provide a functional, break-bulk (non-container) shipping facility for your seaports.

The lot is fully compatible with all CDK3-SP lots and can be used with the CDK3-SP Container Seaport. This is an "additional seaport" and does not replace any exisiting version.

In addition to using the same random & timed rail, truck & cargo props used on the Row Warehouse lots, this seaport introduces two new variations of the classic cargo ships that first appeared on the Vandy Shipping lot. With the new versions, the ships's holds are open and the ship's cranes are busily loading and off-loading cargo.



IMPORTANT
This seaport lot uses the PEG-CDK3-SP Master Seaport Controller (Version 1.04 or above) and can not be used with the BSC seaport controller, or any other seaport mod or controller... or any seaport lot not currently supported by the PEG Master Seaport Controller. Please read the included readme file for full details.

 

RAIL CAPACITY
Unlike the game default seaport, this new version allows a direct rail connection and can also receive freight by rail directly from Freight Stations or industry located adjacent to rail. This is accomplished by forcing the Seaport to also act as a Freight Station, converting rail freight to truck freight when it arrives.  Whereas the previous CDK Seaport version required that the lot be coded as a freight station, this new version does not. All the "mojo" is now handled in the transit enabling properties... which have also been streamlined to minimize associated traffic congestion.


** The lot is transit enabled for road connections to the driveways and a rail connection .  The lot must have road access in order to function properly but the driveways do not actually have to be connected. However, the rail must be connected to the lot for the rail connectivity to function.  Rail connectivity is optional. The Seaport will still function properly for truck traffic if you choose not to connect the rail.

INSTALLATION
If installing from the ZIP archive, download the file to your C:\My Documents\SimCity 4\Plugins folder. Then, unzip the file making sure that the "Use Folder Names" option is selected. The product will be installed in a ZZZZ_PEG SEAPORTS sub-folder directly under the Plugins folder. THE FILES MUST BE INSTALLED TO THIS FOLDER.

Once installed, the product will be ready to use in your game. Lots will appear in the menu listed in the above Product Details section.
** After installation, you can either delete the downloaded ZIP file or archive it outside of your Plugins folder satructure.

** Due to the sensitive nature of seaport lots, this lot can not be copied, reused, referrenced  or redistributed in any manner.



The lot is a CDK3 lot and therefore must be plopped over water. It has been designed to have the inner most 3 rows of tiles plopped fully on dry land. 

The 2 driveway entrances are transit enabled as roads. They will connect automatically to an existing road network.  If the road network is added after the lot is plopped, you can connect them manually using roads. These connections are only visual. They are not required for lot functionality. However, the lot must have road access in order to function.

The central rail connection on the front (inland) side is also transit enabled for rail. This must be connected to your rail network for the seaport to accept  freight via rail. Additionally, you must also have a street/road/avenue in front of the lot where the rail connects to the lot. The best method is to simply have street/road/avenue running the length of the lot.

The two rail connections on the sides of the lot are not transit enabled. They are only visual and intended for use with future, seport expansion lots.

The lot must be plopped with the 3 inner rows of tile on dry land. It is often helpful to use the CDK Leveler tool before plopping CDK lots.
