PRODUCT DESCRIPTION


 

Several years in the planning stage... about 8 weeks in the making...  the Next-Gen functional game Seaport is finally here.


The CDK3-SP Container Seaport is a functional seaport that directly replaces the game's default seaport. This is the first and core piece of a series of CDK3 Seaport (CDK3-SP) modular lots that will allow you to build & expand your seaport area, both functionally & visually, to almost any size and the way you want it to be. An associated, functional Container Freight Rail Yard is also under development.

This Seaport lot has all the same properties, capacities and costs of the game-default seaport.  The lot is a different size (10x5)  so any instances of the game default seaport in your cities will have to be replaced. However, the good news on that is that the old seaports will remain functional until you get around to replacing them. You do not have to bulldoze them all before installing the new Seaport.


IMPORTANT
Due to the limiting nature of the common seaport controller file used by all functional seaport lots, no other custom seaport lots, seaport mods or seaport controller files can be used with this seaport. The only exceptions are the PEG CSK2 Small Port... and an additional CDK3-SP "Break-Bulk" Freight Seaport that is under development.  Please read the included readme file for full details.

 
The new Seaport lot uses a gazillion new custom props and most of them are timed to change and/or move throughout the day. Ships arrive and depart twice a day. The booms of the large Post-Panamax cranes raise to allow easy clearance for the ships when arriving/departing... then lower to load & unload containers throughout the day. The yard crane switches from one staging area to another, the stacks of containers grow larger & smaller... and the rail and truck traffic change & move around.


RAIL CAPACITY
Unlike the game default seaport, this new version allows a direct rail connection and can also receive freight by rail directly from Freight Stations or industry located adjacent to rail. This is accomplished by forcing the Seaport to also act as a Freight Station, converting rail freight to truck freight when it arrives.  Whereas the previous CDK Seaport version required that the lot be coded as a freight station, this new version does not. All the "mojo" is now handled in the transit enabling properties... which have also been streamlined to minimize associated traffic congestion.


** The lot is transit enabled for road connections to the driveways and a rail connection .  The lot must have road access in order to function properly but the driveways do not actually have to be connected. However, the rail must be connected to the lot for the rail connectivity to function.  Rail connectivity is optional. The Seaport will still function properly for truck traffic if you choose not to connect the rail.


*** The lot is located in the Water Transportation menu in place of the game-default Seaport icon.

INSTALLATION
If installing from the ZIP archive, download the file to your C:\My Documents\SimCity 4\Plugins folder. Then, unzip the file making sure that the "Use Folder Names" option is selected. The product will be installed in a ZZZZ_PEG SEAPORTS sub-folder directly under the Plugins folder. THE FILES MUST BE INSTALLED TO THIS FOLDER.

Once installed, the product will be ready to use in your game. Lots will appear in the menu listed in the above Product Details section.
** After installation, you can either delete the downloaded ZIP file or archive it outside of your Plugins folder satructure.
DEVELOPER NOTES
EXTREMELY IMPORTANT:

No other functional seaport lots can be used with this seaport lot... except the PEG CSK2 Small Port. All other seaports lots, seaport mods  and seaport controller files must be removed from your cities and uninstalled from the game.

Exception: This lot directly replaces the game-default seaport. The player does not have to remove these from their cities prior to installation of this new version. However, they will need to bulldoze and replop them at their convenience.


* The game uses a common controller file for all seaport lots.  This controller file supports only 3 seaport lots. This seaport lot and the PEG CSK2 Small Port are two of the seaport lots supported... with a 3rd CDK3 seaport currently under development.

Additional seaport lots can theoretically be added... but would require special & complex handling of the controller file and/or mandatory installation of all supported seaport lots. Any error or omission in these limitations, would result in the game crashing with the permanent loss of all cities containing the seaports.  Its just not worth the risk.


** Due to the sensitive nature of seaport lots, no development group or individual developer will be allowed to copy, reuse or redistribute this lot in any manner. Players are permitted to use and modify this lot for their own use... but do so at their own risk.




The lot is a CDK3 lot and therefore must be plopped over water. It has been designed to have the inner most 3 rows of tiles plopped fully on dry land. 

The 2 driveway entrances are transit enabled as roads. They will connect automatically to an existing road network.  If the road network is added after the lot is plopped, you can connect them manually using roads. These connections are only visual. They are not required for lot functionality. However, the lot must have road access in order to function.

The central rail connection on the front (inland) side is also transit enabled for rail. This must be connected to your rail network for the seaport to accept  freight via rail. Additionally, you must also have a street/road/avenue in front of the lot where the rail connects to the lot. The best method is to simply have street/road/avenue running the length of the lot.

The two rail connections on the sides of the lot are not transit enabled. They are only visual and intended for use with future, seport expansion lots.

The lot must be plopped with the 3 inner rows of tile on dry land. It is often helpful to use the CDK Leveler tool before plopping CDK lots.
