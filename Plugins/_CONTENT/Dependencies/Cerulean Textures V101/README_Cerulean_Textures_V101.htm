Cerulean Textures V101

 

This is a collection of textures built into a DAT file for use with custom lots.

 

 

Version Notes :

V1 : Initial release of 26 lot base textures.

V2 : Added Readme File

V3 : Fixed Instance Ids (all textures show up in game now!)

V4 : Added 54 New BASE Textures

V5 : Added 4 new textures. Edited ALL INSTANCE IDs. Any lots made before release 5 will not work, they must be edited.

Version 5 Notes : With the discovery of the wealth level variable inside the instance IDs of all Texture files by myself and members of the BSC Team, the original and the V3 IIDs were deemed corrupt. Much research was needed which was initially not understood, but is now understood. All of the textures in this Texture Pack are now set up to work no matter what wealth level you set your lot to. You should understand that it is still possible for individual textures to become corrupt if someone uses the same IIDs in a future lot release that I have used in this file. Geoffhaw and the BSC are working hard to make sure this doesn’t happen by creating a list of all IIDs used in lot textures, so check the BSC threads for more information on this. If you ever experience any problems with this please contact me personally at www.SimTropolis.com where my name is cerulean.
