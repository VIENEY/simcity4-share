 SFBT Essentials - December 2010 version
The SFBT Essentials contain the custom UIs that every SFBT lot uses, as well as shared textures and prop packs. The file will be updated from time to time, so please check back to this download once in a while.

These are the files that are contained in the installer. Some of them might be installed on your system already, as they were included in older downloads. Since we update them from time to time, it is advisable to run the SFBT Essentials installer again to overwrite the old files, if a lot doesn't work properly.

SFBT Tree Props (Andreas, see below) various prop families that place trees on lots

The following files will determine the type of trees that show up on various lots that support this feature. You can install as many as you want, however the trees will mix randomly then. For a uniform look, it is suggested to install only one file at once. Please note that you have to install the tree props by Cycledogg or Pegasus, otherwise, brown boxes will appear. Note: If you have the SFBT Street Tree Mod installed already, you don't need to install the tree prop families again!

  • SFBT_CP_Street_Deciduous_Trees.dat - a mix of various deciduous trees by Cycledogg (oaks, elms, cottonwood)
  • SFBT_CP_Street_Ponderosa_Pines.dat - various ponderosa pines by Cycledogg
  • SFBT_CP_Street_Shore_Pinyon_Pines.dat - various shore and pinyon pines by Cycledogg
  • SFBT_PEG_Street_Pines.dat - a mix of pine trees by Pegasus
  • SFBT_Maxis_Street_Deciduous_Trees.dat - the default Maxis deciduous trees
  • SFBT_Maxis_Street_Palm_Trees.dat - the default Maxis palm trees
  • SFBT_Maxis_Street_Pine_Trees.dat - the default Maxis pine trees

If you want to use Cycledogg's or Pegasus' trees, you also have to install the following files. The files described above will use trees from all packs by Cycledogg that are listed below, so make sure these are installed properly. If you don't install the dependencies, brown boxes will show up on your lots.

  • BSC MEGA Props - CP Vol01.dat
  • CPT_No8_TreeModelsPartOne.dat and CPT_No9_TreeModelsPartTwo.dat
  • PEG_MTP_RESOURCE_vol1.dat
 Compatibility
This download is only compatible with SimCity 4 incl. the "Rush Hour" expansion pack or SimCity 4 Deluxe.
 Installation

With the installer

Double-click the installer file to start the installation. The program will guide you through the process. You can change the default installation path, but we we strongly recommend installing into the suggested folder in order to make future updates and additions easier.

From a ZIP archive

Extract all files (excluding this readme file and the images) to "\My Documents\SimCity 4\Plugins". Make sure your ZIP program supports the creation of subfolders (look for a "keep paths" or "use subfolders" option). It is wise to create a new folder for each download to avoid confusion.

Uninstallation

For uninstalling the files, simply bulldoze every lot from this download in your cities and delete the files from the plugins folder afterwards. You can also uninstall only the growable or ploppable version by deleting the corresponding lot file.
