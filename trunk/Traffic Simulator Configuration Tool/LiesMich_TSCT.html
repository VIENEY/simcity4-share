Traffic Simulator Configuration Tool von Stefan79 and z

Das Traffic Simulator Configuration Tool ist ein Programm, mit dem Du Deine Traffic Simulator und Data View-Plugins aus dem NAM nach Deinen Wünschen einstellen kannst. Dies kann entweder während der Installation des NAM oder jederzeit danach geschehen.
 Screenshots

 Informationen

Vollständige Informationen zur Benutzung des Traffic Simulator Configuration Tool können im beiliegenden Benutzerhandbuch nachgeschlagen werden, welches über das Hilfe-Menü (siehe Screenshot oben) erreichbar ist. Benutzerer älterer NAM Traffic Simulator Plugins sollten sich vor allem den Abschnitt Verwenden von Traffic Simulator Configuration Tool, um ältere NAM Traffic-Simulator Plugins zu emulieren durchlesen. Dieser Abschnitt beschreibt die besten Einstellungen, um ältere Traffic Simulatoren nachzubilden.
 Kompatibilität

Dieser Download ist ausschließlich mit SimCity 4 inkl. dem Add-On "Rush Hour" bzw. mit SimCity 4 Deluxe kompatibel, und nur für die Verwendung mit Simulator Z oder den aktuellen NAM Unified Traffic Simulatoren (NAM Traffic Simulatoren) gedacht.
 Installation

Windows: Der Installer wird mit einem Doppelklick gestartet und führt automatisch durch den weiteren Installationsverlauf. Selbstverständlich kann der vorgeschlagene Ordner jederzeit abgeändert werden, jedoch empfehlen wir, den Standardordner beizubehalten, um später eventuelle Updates leichter installieren zu können.

MacOS: Die Dateien können mit dem Programm FileJuicer entpackt werden.

Deinstallation: Um das Programm zu deinstallieren, sollten Windows-User das Programm Uninstall.exe aus dem Programmordner aufrufen oder den entsprechenden Eintrag in der Systemsteuerung auswählen. Mac-User können einfach den kompletten TSCT-Ordner löschen.
