INSTALLATION
If installing from the ZIP archive, download the file to your C:\My Documents\SimCity 4\Plugins folder. Then, unzip the file making sure that the "Use Folder Names" option is selected. The product will be installed in an appropriate folder under the PEGPROD sub-folder which will be created if it does not already exist on your system. Once installed, the product will be ready to use in your game. Lots will appear in the menu listed in the above Product Details section.
** After installation, you can either delete the downloaded ZIP file or archive it outside of your Plugins folder satructure.
