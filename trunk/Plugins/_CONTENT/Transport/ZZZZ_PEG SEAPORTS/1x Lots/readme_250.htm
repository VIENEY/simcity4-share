PRODUCT DESCRIPTION


 

Note - Version 1.10 updated to remove the container props from all but the 1x Dock lot to give the lots more flexibility & allow them to be used in all areas of your seaport complex. A 1x Break-Bulk 1x Dock lot has also been added... and the bus stop has been remodeled to a more generic, industrial style. 

After installation, all exisiting lots in-game, except the 1x Container Dock,  should be replopped to update them visually.


The CDK3-SP 1x Lots is a collection of 5, single tile wide lots that have been designed to be used as "filler" between the larger lots of the Seaport series of lots. The lots are 5 tiles deep and blend seamlessly with other CDK3-SP lots. The lots are simple eye candy... cheap to plop and no monthly costs. Each lots uses a very small amount of water & power and creates a bit of garbage.

The 1x Dock lot is a simple extension of the container processing complex. Random containers will appear up on the dock area while trucks and a container "stacker" load and unload containers. The various props are timed and random... so plopping several of these together shouldn't be too visually repetitive.

The 1x Dock Ramp is a variation of the 1x Dock lot that features an access ramp large enough for vehicle traffic.

The 1x Dock Stairs is a variation of the 1x Dock lot that features a set of stairs.

The 1x Dock Parking Lot is very similar to the 1x Dock except that it has a parking area for Seaport employees. Instead of  trucks and a stacker, this lot feature one of several randomly selected container rail cars that appear during the day. The parking lot driveway is transit enabled to form a visual connection only. 

The 5th lot in this pack is the Seaport Bus Stop. It is visually similar to the Parking Lot but has a bus stop at the front of the lot.  This is a fully functional bus stop with all the same costs & properties of a normal bus stop. The lot does not have any jobs but still requires road access for the buses. This lot is packaged as a separate file so the player can easily remove if desired.

Note -  By default, icons for the Seaport Bus Stop will appear in both the Misc Transportation and the Water Transportation menus. If you do not want a duplicate icon in the Water Transportation menu, delete the PEG-CDK3-SP_Dock-BusStop_WT-IconDupe_100.Dat  file included with the installation. This file merely contains a copy of the building exemplar file that duplicates the icon menu in the Water Transportation menu.

* Some CDK3-SP lots may be shorter than the ships that tie up to them. These 1x "filler" lots are ideal for adding space between these larger lots and keeping the various ships from getting too close to each other.

INSTALLATION
If installing from the ZIP archive, download the file to your C:\My Documents\SimCity 4\Plugins folder. Then, unzip the file making sure that the "Use Folder Names" option is selected. The product will be installed in an appropriate folder under the PEGPROD sub-folder which will be created if it does not already exist on your system. Once installed, the product will be ready to use in your game. Lots will appear in the menu listed in the above Product Details section.
** After installation, you can either delete the downloaded ZIP file or archive it outside of your Plugins folder satructure.
DEVELOPER NOTES

The lot is a CDK3 lot and therefore must be plopped over water. It has been designed to have the inner most 3 rows of tiles plopped fully on dry land. 

The Parking Lot lot has a driveway entrance that is transit enabled to make a visual connection only. The lot itself is not transit enabled. It will connect automatically to an existing road network.  If the road network is added after the lot is plopped, you can connect it manually using streets. This connection is only visual. It is not required for lot functionality. However, the lot must have road access in order to function properly as it has jobs..

The central rail connection on the front (inland) side is also transit enabled for rail. This must be connected to your rail network for the seaport to accept  freight via rail. Additionally, you must also have a street/road/avenue in front of the lot where the rail connects to the lot. The best method is to simply have street/road/avenue running the length of the lot.

The two rail connections on the sides of the lot are not transit enabled. They are only visual and intended to create a visual railroad spur line running the length of of all the adjacent lots.

The lot must be plopped with the 3 inner rows of tile on dry land. It is often helpful to use the CDK Leveler tool before plopping CDK lots.
