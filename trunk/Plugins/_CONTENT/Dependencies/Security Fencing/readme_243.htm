PRODUCT DESCRIPTION


  

This resource pack is a collection of chain link fence props that can be used on any type of lot that requires security fencing.  The props in this pack may be used by other developers to enhance the appearance of their lots.

The pack contains two styles of fencing (with or without barb wire) and includes extra items such as diagonal fence sections, corner sections, pedestrian gates and automatic rolling styled gates.


* This pack will be a dependency for future PEG lots.




 
** This dependency pack mod has no dependencies.

INSTALLATION
If installing from the ZIP archive, download the file to your C:\My Documents\SimCity 4\Plugins folder. Then, unzip the file making sure that the "Use Folder Names" option is selected. The product will be installed in an appropriate folder under the PEGPROD sub-folder which will be created if it does not already exist on your system. Once installed, the product will be ready to use in your game. 

** After installation, you can either delete the downloaded ZIP file or archive it outside of your Plugins folder structure.
DEVELOPER NOTES
The ZIP file should create a new MODs folder within the PEGPROD sub-folder of your Plugins folder. This is the reccomended location for this file but it should work properly if located anywhere within your PLugins folder structure.
