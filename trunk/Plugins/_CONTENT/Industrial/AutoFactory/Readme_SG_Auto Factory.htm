 Compatibility
This download is only compatible with SimCity 4 "Rush Hour" or SimCity 4 Deluxe.
 Installation

Auto Installation

Double-click the installer file to start the installation. The programm will guide you through the process. You can change the default installation path, but we strongly suggest to keep the standard folder in order to make future updates and additions easier.

MAC Users

There will be a Mac compatable version of this file available in the BSC Mac Archive.

Un-Installing

To un-install the files, simply bulldoze every lot from this download in your cities and delete the files from the plugins folder afterwards. You can also un-install only the growable or ploppable version by deleting the appropriate lot installed. The Plop or Reward lot will start with "SG_"; Growables start with the initials of the lot style they are.

 Notes about Nightlighting Effects
In order to see the nightlighting, you must install the EP1-Update 1 for Rush Hour/Sim City 4 Deluxe, and then afterwards, install the SC4-Update for nightlighting (note: you have to be logged in in order to see the download link). If this update isn't installed, you won't see any lighting effects on custom BAT buildings.
